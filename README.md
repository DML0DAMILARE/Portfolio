# Oni Oluwadamilare Josiah — Portfolio

A dark, editorial portfolio site for a Financial & Product Risk Analyst.

## Files
- `index.html` — the full single-page portfolio
- `vercel.json` — Vercel deployment config

## Deploy to Vercel (3 steps)

### Option A — Vercel CLI (fastest)
```bash
npm install -g vercel
cd portfolio/
vercel
```
Follow the prompts. Your site will be live at a `.vercel.app` URL in under 60 seconds.

### Option B — GitHub + Vercel Dashboard
1. Push this folder to a GitHub repo
2. Go to https://vercel.com/new
3. Import the repo → Vercel auto-detects `vercel.json` → click Deploy

## Customising Projects
In `index.html`, find the `<!-- PLACEHOLDER CARDS -->` comment and replace the `href` and content with your real GitHub repo links and descriptions.

## Custom Domain
In Vercel dashboard → your project → Settings → Domains → add your domain.
